# Мониторинг безопасности (автоматизированное реагирование на угрозы)

Итоговое домашнее задание по Python. Скрипт анализирует логи веб-сервера, ищет подозрительные IP и имитирует блокировку.

## 📁 Состав проекта

- `threat_monitor.py` — основной скрипт
- `access.log` — файл с логами (пример)
- `threat_report.csv` — отчет после анализа (создается автоматически)
- `top_ips.png` — график топ IP (создается автоматически)
- `vt_cache.json` — кэш с результатами проверок (создается автоматически)
- `README.md` — этот файл
